A formula $F$ is a conjunction of [Clauses](Clause.md).
It is _satisfiable_ if there exists an assignment satisfying all clauses, otherwise it is _unsatisfiable_.
Formula are defined in [Conjunktive Normalform](Conjunktive%20Normalform.md)(CNF) and generally also stored as such. Any propositional formula can be transformed into CNF using the [Tseitin Transformation](Logik/Tseitin%20Transformation.md).

We denote the [set](Mengen.md) of [Variables](Variable.md) occurring in a formula $F$ by $Var(F)$. Similiary the set of literals is denoted by $Lit(F)$.

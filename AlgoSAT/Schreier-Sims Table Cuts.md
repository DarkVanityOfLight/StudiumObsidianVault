
In [Lex-leader constraints](Total%20Order%20and%20Lexicographic%20Order.md) usually the first positions are the most expressive, the first position of a lex constraint yields a binary clause
$$\ell \le \varphi(\ell) \equiv (\neg \ell \lor \varphi(\ell))$$
Naturally we want to derive as many such binary constraints as possible.

Given a CNF $F$ and a group $\Gamma$ of [Semantic Symmetries](Semantic%20Symmetry.md) of $F$.

1. Pick some $\ell\in Lit(F)$
2. Let $O$ be the [Orbit](Orbit.md) of $\ell$ in $\Gamma$.
3. If we order $\ell$ to the front of our total order, we know that $\ell$ must be smaller or equal to all $\ell'\in O$
4. That is we can produce the constraints $\ell \le \ell'$ for all $\ell'\in O$
5. $\ell \le \ell' = (\overline \ell \lor \ell')$

If we attempt to repeat this for another literal, the symmetries used to determine the orbit of $\ell_2$ may need to move $\ell$.

> Given a point $\ell\in\Omega$, we define the _pointwise stabilizer_
> $$\Gamma_{(\ell)} = \{ \varphi\in\Gamma | \varphi(\ell) = \ell \}$$
> Similarly, for a [set](Mengen.md) $L \subseteq \Omega$ we define the setwise stabilizer
> $$\Gamma_{\{L\}} = \{ \varphi\in\Gamma | \varphi(L) = L\}$$

1. Initialize an empty parital order $\prec$ and $L=\emptyset$
2. Pick some $\ell\in Lit(F)\setminus L$.
3. Let $O$ be the orbit of $\ell$ in $\Gamma$.
4. Order $\ell$ after $L$ but before all $Var(F)\setminus L$.
5. Produce the constraints $\ell \le \ell' = (\overline \ell \lor \ell)$ for all $\ell'\in O$
6. Set $\Gamma = \Gamma_{(\ell)}$ and $L = L \cup\{\ell\}$, and go back to (2)

## Computing Pointwise Stabilizers

Given a permutation group $\Gamma \le Sym(\Omega)$ and a point $\ell\in\Omega$, compute the generators for the stabilizer $\Gamma_{(\ell)}$
- Let $\Gamma = \langle S \rangle$ be generated by a set $S$
- Let $O$ be the orbit of $\ell$
- A _transversal_ $T_\ell = \{t_x | x\in O\}$ is a set such that
  $$t_x(\ell) = x \text{ and } t_\ell = \ell$$
- For each $\varphi\in\Gamma$ we denote with $\overline\varphi \in T$ it's representative in $T$

> Let $\langle S\rangle =\Gamma \le Sym(\Omega)$ and $\ell \in\Omega$. The group $\Gamma_{(x)}$ is generated by the set
> $$\{ts(\overline{ts})^{-1} | t\in T_\ell, s\in S\}$$


## Proving Schreier Cuts

For a given formula $F$ and one of it's literals $\ell$, we derive all binary clauses $\overline \ell \lor \ell'$ with $\ell'$ in the orbit of $\ell$ using the syntactic symmetry $\omega$ mapping $\omega(\ell) = \ell'$.

Let $L$ be the binary clauses added so far. We show that
$$F \land L \land \{\ell\} \land \{\overline \ell'\} \vDash (F\land L \land \{\overline \ell, \ell'\})|\omega$$
Every clause in $F$ is entailed, since $\omega$ is a symmetry of $F$. Every other clause $C \in L \cup \{\overline \ell, \ell'\}$ is entailed by the unit clause $\{\overline \ell'\}$, since $\overline \ell \in C$ and $\omega(\ell) = \ell'$.

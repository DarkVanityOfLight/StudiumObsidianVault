# Ganze Zahlen
Für jede natürliche Zahl 1, 2, ... lässt sich eine Gegenzahl konstruieren, sodass die Summe aus Zahl und Gegenzahl gerade Null ergibt. Sie wird durch Hinzufügen eines negativen Vorzeichens mit -1, -2, ... bezeichnet. Damit wird die Menge der natürlichen Zahlen so vergrößert, dass damit auch Gleichungen wie $3 + x = 0$ gelöst werden können.

> Die Menge der ganzen Zahlen ist die Menge $$Z = \{...; -2; -1; 0; 1; 2;...\}$$

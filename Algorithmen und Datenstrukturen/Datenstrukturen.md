## Beispiel: Arrays

Auf ein Array A der Größe $n$ können wir wie folgt zugreifen:

### Auswertung:
Gegeben $1 \leq i \leq n$ gibt das $i$-te Element zurück

$A[i]$ -> Der Aufruf $A[i]$$ gibt $a_i$ zurück

### Zuweisung:
Gegeben $1 \leq i \leq n$ und $x$, steze das $i$-te Element auf $x$

### Intialisierung

Den Speicherbereich für ein Array zu reservieren(Allokation) ist in konstanter Zeit möglich

__Achtug__: Ohne besondere Initialisierung können wir keine Annahmen über den Inhalt machen!

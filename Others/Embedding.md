
$f: A \to B$ is an _embedding_ if $f$ is an [injective](Injektiv.md) [Morphism](Morphism.md) which induces an [Isomorphism](Isomorphism.md) from $A$ to the substructure $f(A)$ of $B$.

An embedding does not merely inject elements. It says:
$A$ and its image $f(A) \subseteq B$ are structurally identical.
All structure on $A$ is exactly the structure inherited from $B$.
So conceptually:
$$A \text{ sits inside } B \text{ without losing or changing structure.}$$
